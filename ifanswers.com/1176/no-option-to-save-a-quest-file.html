Hi everyone.

This is my first time using Quest, so I apologize if this question has been asked and/or answered before.

I spent about 5 days coding a storybook on the offline Quest editor, and when I finalized everything this afternoon, I clicked the publish option to finally upload it. However, when I went to save it from a .aslx file to a .quest file, it didn't give me an option to save at all.

This looks like a bug in Quest where the window isn't displaying fully when you're using high DPI settings.

As a workaround, go into your display preferences and set display scaling to 100%. You should then be able to see the Publish button on that window.

Alternatively, just hitting the Enter key when that window is open might work.
