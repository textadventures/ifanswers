You can search for games on the Interactive Fiction Database by tag. Example: Here's a search for entries that have been tagged "I7 source available": http://ifdb.tads.org/search?searchfor=tag:I7+source+available

Here's the full list of tags: http://ifdb.tads.org/showtags

If searching IFDB doesn't help, you can create a poll on IFDB, describe what kind of IF you are looking for, and ask for recommendations.
