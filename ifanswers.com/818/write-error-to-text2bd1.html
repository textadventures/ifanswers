Write error to text

0 votes
71 views
asked Jun 21 in Authoring by Aika

Hi,

I have an error (P32 wrong kind of thing to decide on) that occurs during a part of the game where menus are up and afterwards the screen is cleared, meaning I don't get to see the details of the error.

Is there some way I can write the details of the error to a text variable or some other way I can view the details of the error?

Cheers!

You can turn on a transcript before it happens, or if you are playing it from within Inform, then it will be shown in the transcript tab.
