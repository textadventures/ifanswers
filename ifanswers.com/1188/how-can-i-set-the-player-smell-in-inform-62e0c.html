How can I set the player smell in Inform 6?

0 votes
74 views
asked Sep 17 in Authoring by Oreolek (1 point)

I can set the player description using, well, player.description

But how can I set the smell? As in > smell self.

  • inform6
  • programming

1 Answer

0 votes
answered Sep 19 by anonymous

First of all, you would need to create a new player object.

Then you would set a newplayer.smell property, or a newplayer.before[ Smell: "some text here."], attribute.
