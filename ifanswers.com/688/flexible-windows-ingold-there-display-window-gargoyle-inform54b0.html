With Flexible Windows by Jon Ingold, is there a way to make the text display at the top of the window, even in Gargoyle? (Inform 7)

0 votes
1,020 views
asked May 28 in Authoring by bg (692 points)

If you create a text window, can you make the text align with the top of the window instead of the bottom, even in Gargoyle?

related to an answer for: In Inform 7, how do you create a text window using Flexible Windows by Jon Ingold?

No, that's just how Gargoyle is.

But if you know exactly how many lines of text you will be printing, you could spawn a window of exactly that height positioned to the top, which would appear as if it were top aligned. Similarly you can centre text by using extra wide 'margin' windows.
