Can you assign other names in browser Quest

0 votes
42 views
asked May 8 in Authoring by anonymous

I am currently attempting to make a game in the browser version of the Quest software and I can't figure out how to assign other names to objects. The tutorial mentions an Object tab where you can add these additional names, but it does not appear to be present in the version I am using. Any suggestions?

Turn off Simple Mode from the Settings button at the top of the screen, then you'll see the Object tab.
